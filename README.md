# Bitcoin Converter

Frontend technical interview task. The goal is to create an application for converting Bitcoin to USD, EUR, GBP.

## Requirements for application

 1. Create form with input field for BTC value.
 2. Add dynamically currencies (USD, EUR, GBP).
 3. If currency removed allow to add it back to the results table.
 4. Currencies table should have formatted currency. 
 5. Currency rates updated each minute

## Run project

    `npm install`
	`ng serve`

Navigate to `http://localhost:4200/`.

## Run tests

    `ng test`
